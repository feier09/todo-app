# 我的待办清单 📝

一个简洁现代的待办事项 Web 应用，纯 HTML + CSS + JavaScript 实现，数据保存在浏览器本地存储。

## ✨ 功能特性

- **任务管理**：添加、编辑、删除任务
- **优先级**：高 / 中 / 低 三档，彩色标识 + 自动排序
- **完成状态**：点击圆形复选框标记完成，文字显示删除线
- **数据统计**：实时显示未完成与已完成数量
- **批量清理**：一键清空已完成 / 全部清空（带美观确认弹窗）
- **本地存储**：刷新页面、关闭浏览器数据不丢失（localStorage）
- **响应式设计**：手机端体验友好

## 🚀 本地预览

直接用浏览器打开 `index.html` 即可。

或在项目目录启动一个静态服务器：

```bash
npx serve .
```

## 🎨 技术栈

- 原生 HTML5 / CSS3 / JavaScript（无任何框架与依赖）
- localStorage 持久化
- 自定义 SVG 图标 + CSS 动画
